# Yummy Chat

코딩셰프 조금 매운맛 실습 - 채팅 앱

### 주요 실습기능

* Firebase 연동 및 사용법
* Mixins
* Stream / stream builder

### 특이사항

* 2022.06 기준 최신 문법 (Flutter 3.0.3)에 맞도록 수정
* 경고 및 중복 코드를 가능하면 줄이는 방향으로 리팩토링

### 작성환경 (2022.06 기준)

* macOS 12.4 Monterey
* Android Studio Chipmunk (2021.2.1, patch 1)
* Flutter 3.0.3
